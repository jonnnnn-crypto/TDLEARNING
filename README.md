# TDLEARNING

TD LEARNING (Technology Digital Learning) adalah platform pembelajaran teknologi berbasis komunitas yang dirancang untuk membantu pelajar terutama di Lampung Barat dalam mempelajari teknologi digital secara terstruktur, interaktif, dan realtime.

## Features
- Komunitas belajar
- Diskusi realtime & Voice channel
- Upload materi pembelajaran
- Dashboard admin
